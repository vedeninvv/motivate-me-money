# Motivate-Me-Money
#### Сервис для повышения мотивации выполнения заданий за счет геймификации

1. Пользователь создает задания, которые хочет выполнить, и выбирает для них сложность
2. За выполнения заданий получает монетки
3. Покупает специальные предметы, дающие бонусы
4. Благодаря бонусам число получаемых монеток растет
5. Желание скопить как можно больше и соревнование с другими пользователями мотивирует выполнять задания

## Backend:
* Spring
* База данных: Postgres
* Схема аутентификации: Basic
